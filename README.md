## コマンド
yarnを使っています
### yarn dev

### srcフォルダ内の各フォルダの役割について
- components
ヘッダーやフッダーなど、各ファイル内に使用されるパーツを格納します。
- hooks
カスタムフックを格納します。
- lib
使用中のライブラリに関する設定などを格納します。
- types
TypeScriptの型定義を格納します。
- utils
ページやコンポーネント間で使い回す変数情報などを格納します。

### 命名規則
@typescript-eslint/naming-conventionで管理。
ルールについては.eslintrc.json参照
